KEY FILES TO CUSTOMIZE
================================================================================

1. COLORS: assets/css/style.css (lines 17-26)
   - Update CSS custom properties for branding

2. LOGO: Panel → Paramètres du site → Logo
   - Upload logo image in Kirby Panel

3. CONTACT INFO: Panel → Paramètres du site
   - Address, phone, email, opening hours, social media

4. PAGES: Panel (/)
   - Create pages using the defined blueprints

5. FONTS: site/snippets/header.php + assets/css/style.css
   - Change font imports and --font-primary variable
